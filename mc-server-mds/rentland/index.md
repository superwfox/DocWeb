# 领地相关 ![right](../image-6.png)

 &nbsp;服务器的领地接入了`QQ`，如果领地有他人踏足，您会收到`QQ`消息提醒，哨兵模式是默认开启的。 更有意思的是您可以通过该系统直接给游客权限，无需上线游戏，这很科幻对吗？ 但是的，我们做到了。
{#anchor}
## 领地功能

### - 哨兵模式 
 当您的领地有陌生人踏足，您会在第一时间收到`QQ`消息提醒。

:::info
注意该功能需要主动与`机器人`发消息开启 这是`QQ`的限制 我们也无能为力
:::

### - 游客权限 
 对于陌生人，禁止所有领地中的交互，包括吃东西，道具使用，红石，以及方块。

### - 领地传送
 领地提供传送功能，位于领地中央最高处，只有拥有领地权限才能看到该传送点，方便您快速到达。

### - 领地数量
 领地并没有数量限制，但是最多提供18个可供传送的槽位。

### - 领地大小
 领地大小没有上限 但最小必须大于 `10*10`。
